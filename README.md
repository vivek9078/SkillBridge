# SkillBridge 🚀

SkillBridge is an AI-powered career guidance platform designed to help students and job seekers learn skills, track progress, explore opportunities, and get personalized career guidance.

---

## 🌟 Features

- 🤖 AI Career Guidance
- 📚 Curated Learning Resources
- ✅ Skill Progress Tracking
- 💼 Internship & Job Opportunities
- 🧑‍🏫 Mentor Support
- 📈 Career Roadmaps

---
